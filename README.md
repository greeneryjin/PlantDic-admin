# 가드너스클럽 관리자 사이트(admin)
관리자가 저장할 가드너스클럽 사이트 


구현 : formlogin, s3 업로드, 식물 사전 CRUD 

사용 언어
```
- JAVA 8
- react
```

사용 기술
```
- springboot
- Mysql
- react
- Querydsl
- spring data jpa
- spring jpa
```

라이브러리
```
- lombok
- gradle
```
